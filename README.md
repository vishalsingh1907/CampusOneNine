# CampusOneNine

> **A modern student platform built with Python and Reflex.**

CampusOneNine is an open-source platform designed to centralize academic resources for college students. The goal is to provide a clean, fast, and scalable platform where students can access notes, assignments, reports, and project resources from a single place.

The project is currently being rebuilt from the ground up using a **Python-first architecture** with **Reflex**, replacing the previous Firebase-based implementation.

## Why Reflex?

The entire platform is being developed using **Reflex**, allowing both the frontend and backend to be written in Python. This simplifies development, keeps the codebase consistent, and enables rapid iteration without switching between multiple programming languages.
